# Lit-Anchor Format — Whately Logic Unit

Lit-anchors are recurring literary anchor handouts for the Whately logic unit. They connect the weekly logic concept to a stable literary text so students can practice reasoning with familiar material rather than relearning plot context each week.

## Purpose

A lit-anchor is not a general literature worksheet. It is a bridge between:

1. the week's Whately logic concept;
2. a focused passage from Shakespeare;
3. guided logical analysis;
4. independent student analysis;
5. a portfolio artifact.

## Required format

Each lit-anchor may be a single self-contained student-facing handout when the work is not being graded by the AI evaluator. When a lit-anchor is intended for AI evaluation, split it into two student-facing products:

1. a lit-anchor reading handout containing the literary passages, purpose, central question, light glosses, and any non-written reading focus; and
2. an anchor worksheet using the same evaluator-safe format as the focus labs: repeated `Student Name` / `Assignment ID` / `Page X of Y` header, stable `Part X QY:` prompt labels, and matching `Part X QY ANSWER BOX` labels.

For Week 1, the anchor worksheet assignment ID is `U1L1AW`.

Required sections:

1. **Purpose**
   - Explain the link between the week's logic lesson and the literary passage.

2. **Central Question**
   - One precise question that frames the logic problem in the play.

3. **Guided Text**
   - Original Shakespeare text only.
   - Use light side-note glosses for difficult words, conceptual cues, or logic reminders.
   - Do not paraphrase the passage in place of the original.

4. **Guided Analysis**
   - Model how to use the week's Whately tool.
   - Identify claim/conclusion, stated reason, hidden assumption, and test when appropriate.

5. **Discussion**
   - Interpretive questions that connect logic and literature.
   - Avoid comprehension-only worksheets.

6. **Independent Text**
   - A second Shakespeare passage for students to analyze on their own.
   - Original text only, with light side-note glosses.

7. **Independent Analysis**
   - Student-facing questions or writing lines.
   - Should create work that can become a portfolio artifact.

8. **Portfolio Artifact**
   - Clearly label the artifact students may save.
   - Connect it to the portfolio category for the week.
   - In split AI-evaluable versions, move this written artifact to the anchor worksheet and give it stable Part/Q labels.

9. **Closing Synthesis**
   - Return explicitly to Whately and the week's logic habit.

## Style rules

- Use original Shakespeare text only for play excerpts.
- Glosses should be light side notes, not modernized replacement text.
- Keep side glosses short and sparse enough that they do not collide. In LaTeX, use small side-note type and tight line spacing for margin glosses.
- Leave visible breathing room before student writing lines; the first line should not sit immediately under the question.
- Use direct student-facing language.
- Keep teacher notes out of the student handout.
- Prefer recurring Macbeth and Julius Caesar examples across the unit.
- For Lesson 1, use Macbeth only because students will watch Macbeth before the unit.
- The lit-anchor should make logic useful for reading literature, not reduce literature to a logic puzzle.

## Lesson 1 model

Lesson 1's Macbeth lit-anchor uses:

- Guided Text: *Macbeth* Act 1, Scene 3, the witches' prophecy and Macbeth's first reasoning.
- Independent Text: Act 1, Scene 7, Macbeth's soliloquy before Duncan's murder.
- Core insight: prediction is not permission; possibility is not command; desire is not proof.
- Portfolio artifact: Macbeth argument skeleton.

## Source-note practice

Every lit-anchor should record:

- play, act, scene, and line range when available;
- source edition or URL used for the public-domain text;
- any adaptation policy. For lit-anchors, the play text should not be adapted, only lightly glossed.

# scope_wr121

Whately *Elements of Logic* curriculum (WR121).

- Student reading PDFs: `units/elements-of-logic-weekN/student-packets/`
- Graded Logic Lab and Lit Example worksheets: `graded-assignments/U#L#LL/` and `graded-assignments/U#L#LE/` (authoritative Krewone triples: `worksheet.tex`, `worksheet.pdf`, `template.json`)
- Student readings and Lit Example readers: `units/elements-of-logic-weekN/student-packets/` and `units/elements-of-logic-weekN/lit-examples/`
- Answer keys: `answer-keys/U#L#LL.md`, `answer-keys/U#L#LE.md`, and Logic Dungeon keys matching `U1F#LD` (same grader format as scope_tenth)
- Logic Dungeon finals: `finals/` (six-character ID pattern **U1F#LD** for all dungeons; four three-week checkpoints through Week 12 and a cumulative final after Week 17)
- **Krewone graded assignments:** flat authority under `graded-assignments/<ID>/`, index `graded-assignments.json`, shared package `latex/grader-worksheet.sty`, generator `scripts/build-grader-templates.py`
- Logic Dungeon plan: `docs/logic-dungeon-assessment-plan.md`
- Format: `docs/logic-lab-format.md`
- **Authoring (new labs / AI handoff):** `docs/logic-lab-authoring-guide.md`
- **Full-course plan:** `docs/full-course-plan/README.md` (complete 19-week Whately sequence)

Week folders: `units/elements-of-logic-week1/README.md` (**U1L1LL**) through `units/elements-of-logic-week19/README.md` (**U1L19LL**).
